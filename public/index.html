Educational OS project

A Python file-system simulator with recovery, caching, and space-allocation strategies.

Inode-based storage, bitmap free-space management (first/best/worst fit), LRU cache with read-ahead, checkpoint-based crash recovery, and corruption detection — all in a single Python codebase with a CLI and a Flask API.

Heads up: the interactive simulator is a Python/Flask application that runs locally. This page is a static overview — clone the repository and follow the quickstart below to run the tool on your machine.

Modules

01

filesystem_core

Inode metadata, block storage, checksum integrity.

02

free_space_manager

Bitmap allocation with first-fit, best-fit, worst-fit, and defragmentation.

03

directory_manager

Hierarchical directories, path resolution, traversal, statistics.

04

file_access_optimizer

LRU cache, predictive read-ahead, access-pattern analysis.

05

recovery_manager

Checkpoints, orphaned-inode detection, crash simulation, repair.

06

disk_simulator

Integration layer tying the modules into a single API surface.

Quickstart

# 1. Install Python deps
$ pip install -r requirements.txt

# 2. Run the interactive CLI
$ python main.py

# 3. Or run the Flask API
$ python web_app.py
#    → http://localhost:5000
# Programmatic usage
from disk_simulator import DiskSimulator

sim = DiskSimulator(1024, 4096)
docs = sim.create_directory("documents")
f    = sim.create_file("readme.txt", 2048, docs)
sim.write_file(f, b"hello")
sim.checkpoint("before crash")
sim.simulate_crash("moderate")
sim.repair()
